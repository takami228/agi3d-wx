agi3d-wx
========
## 必要なソフトウェア  

### Macの場合  

* wxWigets (homebrewからインストール可能)  
* libLBFGS http://www.chokkan.org/software/liblbfgs/  
* CML http://cmldev.net/  
* Boost (homebrewからインストール可能)

$(HOME)/Software/include にCMLを解凍してできたフォルダの中にあるcml/を置く．

### Linuxの場合(まだ動作未確認)  

* wxWigets http://www.wxwidgets.org/  
* BLAS  
* LAPACK  
* Boost
* libLBFGS http://www.chokkan.org/software/liblbfgs/  
* CML http://cmldev.net/
